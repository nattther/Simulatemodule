

Prérequis :

    Un serveur Apache (tel que XAMPP) installé sur votre machine.
    MySQL installé sur votre serveur Apache.

Instructions pour lancer le programme avec XAMPP:

    Téléchargez et installez XAMPP depuis le site officiel (https://www.apachefriends.org/fr/index.html), si vous ne l'avez pas déjà fait.

    Une fois XAMPP installé, localisez le dossier d'installation et ouvrez-le.

    Copiez le contenu de votre projet dans le dossier "htdocs" de XAMPP.

    Démarrez XAMPP et assurez-vous de cocher les cases pour démarrer les services Apache et MySQL.

    Ouvrez votre navigateur web et accédez à l'adresse http://localhost/Test/index.php

    Vous devriez maintenant pouvoir visualiser et interagir avec le programme.
